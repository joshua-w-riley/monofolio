# i-am-joshua.com

A personal website, creative system, frontend project, and archive for Joshua W. Riley.

Built as a minimal, single-page web experience using HTML, CSS, and vanilla JavaScript. The site functions as a central place for selected work, music, updates, contact links, and ongoing creative systems.

## Live Site

https://i-am-joshua.com

## Overview

This project is intentionally small in stack and deliberate in structure.

The site uses hash-based navigation to move between sections without a framework, while keeping each page visually consistent through modular CSS, shared spacing systems, and restrained interaction patterns.

The design direction is minimal, editorial, and systems-oriented. The goal is not to create a traditional portfolio template, but a quiet personal archive that can expand over time.

## Design Notes

The interface is built around restraint, spacing, typography, and rhythm.

Rather than relying on heavy visual effects, the site uses subtle transitions, consistent hierarchy, and quiet interaction states. The footer acts as a small expandable colophon, while the main navigation keeps the experience simple and direct.

The visual system is designed to support future content without requiring a redesign.
